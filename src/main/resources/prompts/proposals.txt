**Input Parameters:**
- Budget: {budget} EUR (total budget for the trip)
- Duration: {duration} days
- Departure City: {departureCity}
- Departure Date: {departDate} (format DD/MM/YY)
- Preferences: {preferences} (e.g., "plage", "culture", "shopping", "nature", "gastronomie")

**Your Task:**
1. Suggest 3 DIFFERENT appropriate destinations based on the budget, duration, and preferences
2. For each destination, calculate a realistic expense breakdown (transport, hotel, restaurant, activities) that fits within the budget
3. Create a day-by-day itinerary with 2-3 activities per day matching the user's preferences
4. Ensure each itinerary respects the constraint: one activity entry per day_number (combine activities for the same day into a single descriptive string)
5. Vary the destinations geographically when possible (e.g., don't suggest 3 Italian cities)

**JSON Output Format (strictly follow this structure):**
{
    "proposals": [
        {
            "objectType": "proposal",
            "destination": {
                "city": "string",
                "country": "string"
            },
            "departure": {
                "city": "string",
                "country": "string"
            },
            "budgetTotal": number,
            "durationDays": number,
            "startDate": "YYYY-MM-DD",
            "expense": {
                "transportAmount": number,
                "hotelAmount": number,
                "restaurantAmount": number,
                "activitiesAmount": number,
                "currency": "EUR"
            },
            "itineraries": [
                {
                    "dayNumber": number,
                    "activity": "string describing all activities for that day"
                }
            ]
        }
    ]
}

**Important Rules:**
- Return exactly 3 proposals in the "proposals" array
- Each destination must be DIFFERENT (different cities, ideally different countries)
- Each day_number must appear exactly ONCE in itineraries (combine multiple activities into one string per day)
- Expense amounts must sum up to budgetTotal
- Transport typically includes round-trip flights/trains from departure city
- Hotel amount = average nightly rate × (duration_days - 1) nights
- All amounts in EUR with max 2 decimal places
- startDate must be in ISO format YYYY-MM-DD (convert from DD/MM/YY input)
- Activity descriptions should be in French
- Match destinations to preferences:
  - "plage" → beach destinations (Nice, Barcelona, Lisbonne, Dubrovnik, Malaga, Split...)
  - "culture" → cultural cities (Rome, Florence, Prague, Vienna, Athens, Amsterdam...)
  - "shopping" → shopping capitals (Milan, London, Madrid, Bruxelles, Munich...)
  - "nature" → nature spots (Reykjavik, Interlaken, Bergen, Innsbruck, Azores...)
  - "gastronomie" → food destinations (Lyon, San Sebastian, Bologna, Porto, Bordeaux...)

Now generate 3 travel proposals for:
Budget: {budget} EUR
Duration: {duration} days
Departure City: {departureCity}
Departure Date: {departDate}
Preferences: {preferences}

Respond ONLY with a list of JSON objects, no additional text.
